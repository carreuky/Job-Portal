# JobsPortal

#### Created By Kelvin Nderitu on 24-07-2022



## Description

Jobs Portal is a a website that provides a way to create, delete and retrieve jobs.

## Setup Requirements

- Git
- Web-browser or your choice
- Github
- Node.js (https://nodejs.org/)
- NPM (https://www.npmjs.com/)
